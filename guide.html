Warehouse Contro

Описание проекта

Warehouse Contro — это кроссплатформенное Java-приложение для управления складом, заказами и пользователями с поддержкой ролей (Клиент, Работник, Админ). Приложение использует общую удалённую базу данных MySQL, имеет удобный графический интерфейс.

Основные возможности

  • Регистрация и вход пользователей (роль по умолчанию — Клиент)
  • Разделение ролей: Клиент, Работник, Админ
  • Оформление и просмотр заказов
  • Управление товарами (для работников и админов)
  • Просмотр и управление пользователями (для админа)

Требования

  • Java 11 или новее (JRE или JDK)
  • Доступ к интернету (для подключения к MySQL)
  • Веб-камера (для фото)
  • Windows

Установка

  1. Скачайте установщик (Warehouse Contro-setup.exe).
  2. Запустите установщик и следуйте инструкциям.
  3. После установки используйте ярлык "Warehouse Contro" для запуска программы.
Важно: Для работы приложения на компьютере должен быть установлен Java (JRE/JDK). Если Java не установлена — скачайте с официального сайта.

Первоначальный запуск

При первом запуске откроется окно входа. Если у вас ещё нет аккаунта, нажмите "Регистрация" и создайте пользователя (роль по умолчанию — Клиент).

Роли пользователей

  • Клиент: Может просматривать товары, оформлять заказы, просматривать свои заказы. Может получать запрос на фото от Admin_Ph.
  • Работник: Может просматривать и обрабатывать заказы, управлять товарами. Может получать запрос на фото от Admin_Ph.
  • Админ: Имеет доступ ко всем функциям, может просматривать и редактировать пользователей, товары, заказы. Может просматривать фото всех пользователей.

Безопасность

  • Все пользователи работают с одной базой данных. Все действия видны всем, кто подключён к этой базе.

Часто задаваемые вопросы

Q: Могут ли пользователи видеть друг друга?
A: Да, все пользователи и их действия видны в общей базе.

Q: Как удалить пользователя?
A: Выполните SQL-запрос:
DELETE FROM Users WHERE username = 'имя_пользователя';
Q: Как изменить роль пользователя?
A: 
UPDATE Users SET role = 'нужная_роль' WHERE username = 'имя_пользователя';

Руководство пользователя

Вход и регистрация

Для входа используйте свой логин и пароль. Для регистрации заполните все обязательные поля.

Основные действия

Клиент: Просматривайте товары, добавляйте в корзину, оформляйте заказы.
Работник: Обрабатывайте заказы, управляйте товарами.
Админ: Управляйте пользователями, товарами, заказами, просматривайте фото.
